Custom File Input

Dosya yükleme ve düzenleme işlemlerini kolaylaştıran inputlardır. Kendine göre bir tasarımı vardır ve bu yüzden görüntüsü standart input'lardan farklıdır. Kullanabilmek için normal bir file input tanımlamamız ve ona "value" özelliği eklememiz gerekiyor, aksi halde standart bir file input olarak kullanılır.

Custom file input'dan detaylıca bahsedelim;
Custom file inputlar, aslında dosyanın kendisini tutmazlar. Onun yerine dosyanın database'deki id değerini tutarlar. Daha sonra dosyanın kendisine erişmek istediğimizde bu id değerini ve bazı hazır fonksiyonları kullanabiliriz. Bu fonksiyonlardan da sonra bahsedeceğim.

Peki dosyaların kendilerini nasıl yükleyeceğimize ve bu custom file input'lara nasıl bağlayacağımıza gelirsek; Öncelikle şunu belirtmek gerekirki dosyayı sistemimize yükleme işlemini panelin kendi FileEditor'ü yapar. Bizde o dosyayı FileEditor'ü kullanarak bizim custom file inputlarımıza bağlarız. FileEditor'e erişebilmek için ise input üzerine mouse ile gelip, çıkacak olan "Değiştir" butonuna tıklamamız yeterli. Karşımıza çıkacak olan FileEditor'ü kullanarak istediğimiz şekilde dosya yükleyip dosyalar üzerinde düzenlemeler yapabiliriz.

FileEditor'den kullanmak istediğimiz dosyayı mouse ile seçtikten sonra enter tuşuna basarsak, veya o dosyaya çift tıklarsak veya editör'ün altında bulunan "Kullan" butonuna tıklarsak, seçtiğimiz dosya bizim custom file input'umuza eklenmiş ve logo'su bağlanmış olacaktır.

Son olarak; dosyamızı yükledik ve custom file input'umuza bağladık; peki bu custom file input'dan seçtiğimiz dosyanın değerini nasıl alabiliriz? Bu işlemi standart şekilde yapabiliriz, input'a standart olarak kullanılan "name" özelliğini veririz. Ve bu custom file input'umuzu bir form içinde kullanırız, ardından dosyamızı seçip form'u post ettiğimizde, PHP aracılığı ile $_POST array'inin içinden input'a verdiğimiz "name" özelliğini arayarak seçtiğimiz dosyanın değerine erişebiliriz.

"readonly" özelliği
Bazı durumlarda dosyamızın başkası tarafından değiştirilmesini istemeyebiliriz. Mesela şöyle bir senaryo varsayalım, sitemizde bir tane "İnsan Kaynakları" formu var ve insanlar bu formu kullanarak bize cv'lerini gönderiyorlar. Bu cv dosyalarınada panel tarafında muhtemelen sadece gözatmak isteriz, yanlışlıkla değiştirmek veya silmek istemeyiz. İşte bu ve bu gibi durumlarda panelimizde dosyalarımızı gösterirken bunlara "readonly" özelliği ekleyerek paneldeki herhangi biri tarafından bilerek yada bilmeyerek dosyanın değiştirilmesine engel olabiliriz.
